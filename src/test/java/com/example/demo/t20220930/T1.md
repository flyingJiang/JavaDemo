# 记录一次沟通

## 数据库
数据查询的过程

客户发送一条查询(sql语句)给服务器;

服务器先检查查询缓存,如果命中了缓存,则立刻返回存储在缓存中的结果,否则进入下一阶段;

服务器端进行sql解析、预处理，再由优化器生成对应的执行计划；

MYSQL根据优化器生成的执行计划，调用存储引擎的API来执行查询；

最后将结果返回客户端
